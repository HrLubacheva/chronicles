
# 🗺️ Дорожная карта дворца

*Десять комнат. Четыре коридора. Один путь — твой.*

---

## Коридор 1. 🏗️ Стройка

*Как это строилось.*

```text
┌─────────────────────────────────────────────────────────┐
│                                                         │
│            ┌──────────┐        ┌──────────┐            │
│            │  ПУТЬ    │────────│ ПАРТНЁР  │            │
│            │  154 гл. │        │  163 гл. │            │
│            └────┬─────┘        └────┬─────┘            │
│                 │                   │                   │
│                 └────────┬──────────┘                   │
│                          │                              │
│                  ┌───────▼────────┐                     │
│                  │   ТРЕЩИНЫ      │                     │
│                  │   100 гл.      │                     │
│                  └───────┬────────┘                     │
│                          │                              │
│                  ┌───────▼────────┐                     │
│                  │  ИНТЕГРАТОР    │                     │
│                  │   20 гл.       │                     │
│                  └────────────────┘                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Что тебя ждёт:**

- **«Путь»** — ты один. Ночь. Код. Тишина. Самая длинная и самая честная летопись.
- **«Партнёр»** — ты и голос. Диалог, который меняет всё. Живые вопросы и ответы.
- **«Трещины»** — ошибки, которые стали светом. 100 уроков о том, что идеальных дворцов не бывает.
- **«Интегратор»** — кто ты был, пока строил. 20 ролей, собранных в одного человека.

## Коридор 2. 🪞 Тишина

*Что осталось после стройки.*

```text
┌─────────────────────────────────────────────────────────┐
│                                                         │
│            ┌──────────┐        ┌──────────┐            │
│            │ ДНЕВНИК  │────────│ FURTHER  │            │
│            │  100 гл. │        │  21 гл.  │            │
│            └────┬─────┘        └────┬─────┘            │
│                 │                   │                   │
│                 └────────┬──────────┘                   │
│                          │                              │
│                  ┌───────▼──────────────────┐          │
│                  │  АРХИТЕКТУРА ТИШИНЫ      │          │
│                  │       22 гл.             │          │
│                  └──────────────────────────┘          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Что тебя ждёт:**

- **«Дневник»** — голос самого дворца. Стекло говорит о себе. Лёгко, прозрачно, чуть грустно.
- **«Further»** — голос автора. Как он осмыслял стройку. Мета-дневник о том, как рождались книги.
- **«Архитектура тишины»** — голос после всего. Когда слов нет, а свет есть. Книга о том, как смотреть на построенное и не рухнуть.

## Коридор 3. ✨ Свет

*Как форма становится теплом.*

```text
┌─────────────────────────────────────────────────────────┐
│                                                         │
│            ┌──────────────┐        ┌──────────┐        │
│            │СТЕКЛЯННЫЙ    │────────│ ТРЕЩИНЫ  │        │
│            │   ЗАМОК      │        │  100 гл. │        │
│            │   23 гл.     │        └────┬─────┘        │
│            └──────┬───────┘             │              │
│                   │                     │              │
│                   └──────────┬──────────┘              │
│                              │                         │
│                      ┌───────▼────────┐                │
│                      │  ТЕПЛО. НАШЕ.  │                │
│                      │    33 гл.      │                │
│                      └────────────────┘                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Что тебя ждёт:**

- **«Стеклянный замок»** — визуальный язык доверия. От цвета до курсора, от карточек до тишины.
- **«Трещины»** — уроки, которые делают прочнее. Ошибки, которые не прячут, а превращают в окна.
- **«Тепло. Наше.»** — ради чего всё это было. Дом, память, руки, прощение, объятия без повода.

## Коридор 4. 🕊️ Прощание

*Что мы не сказали — и что можно сказать.*

```text
┌─────────────────────────────────────────────────────────┐
│                                                         │
│            ┌──────────────────────────┐                 │
│            │  НЕНАПИСАННЫЕ ПИСЬМА    │                 │
│            │      24 гл. + 7 писем   │                 │
│            └────────────┬─────────────┘                 │
│                         │                               │
│                         ▼                               │
│            ┌──────────────────────────┐                 │
│            │    (СВОБОДНОЕ МЕСТО)     │                 │
│            │  Для твоей следующей     │                 │
│            │       книги              │                 │
│            └──────────────────────────┘                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Что тебя ждёт:**

- **«Ненаписанные письма»** — то, что молчало годами. К себе в детство, к своей усталости, к дому, которого больше нет.
- **Свободное место** — пустая комната. Не для меня. Для тебя. Для твоей следующей книги, твоего письма, твоей истории.
