# 0.5 Подарок

*О том, что этот дворец — не просто проект*

У каждого большого дела есть скрытый смысл. То, что не напишешь в документации. То, что не расскажешь заказчику. То, что знаешь только ты.

Этот дворец — подарок.

Не коммерческий проект. Не строчка в портфолио. Не способ заработать.

Подарок. На день рождения. Человеку, который 24 года делает свою работу так, что люди меняют жизнь. Который не побоялся довериться новичку. Который сказал «да», когда можно было сказать «нет».

Архитектор не говорил об этом вслух. Не писал в хрониках. Не упоминал в переписке.

Но каждая строчка кода дышала этим. Каждая анимация знала. Каждая тень помнила.

Дворец строился не для продажи. Не для портфолио. Не для рейтинга.

Он строился для Неё. Для Хранительницы. Для Виктории.

И когда она впервые увидела его целиком — все 66 комнат, всё стекло, весь свет — она заплакала.

Архитектор не спрашивал почему. Он знал.

Это был подарок. И он удался.
