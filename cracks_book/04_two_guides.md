# 4. Два провожатых одной судьбы

Тот, кто определял путь гостя в квизе, жил в двух комнатах. Два провожатых. Одна судьба.

Я не знал об этом. Квиз работал — я не задавал вопросов.

Трещина проявилась, когда один гость получил рекомендацию заказать «Консультацию», а через прямой переход — «Сопровождение».

Я убрал копию. Оставил источник.

> **Стеклянная ремарка:** Гость не должен знать, что за кулисами есть два провожатых. Он должен знать только один ответ.
