# 23.2 Доверие в мессенджерах

*Как вестники в Telegram перестали быть спамом*

Раньше всё летело в один чертог. Квиз. Дары. Обращения. Ошибки. Всё вперемешку. Водопад.

Теперь — два русла.

Первое — для сердца. Только важные обращения. Только то, что требует ответа.

Второе — для памяти. Квиз, дары, всё остальное. С защитой от лавины.

Квиз — не чаще тридцати в минуту. Дары — не чаще двадцати.

Только нужное. Только когда это действительно нужно.

И Хранительница больше не тонет. Она видит главное. Отвечает на важное. Не отвлекается на шум.
