# 47. Спасибо, которое не знало имени

Сообщение «Спасибо за обращение» было одинаковым для всех.

Я добавил переменные. «Спасибо, {{ имя }}. {{ достижение }}».

> **Стеклянная ремарка:** Спасибо с именем — сильнее, чем просто спасибо.
