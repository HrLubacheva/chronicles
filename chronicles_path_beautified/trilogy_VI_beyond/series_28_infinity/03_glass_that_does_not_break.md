# 28.3 Стекло, которое не бьётся

*О том, что держит дворец*

Что держит эти стены?

Не код. Не деньги. Не технологии.

Доверие.

Пока есть люди, которые верят в честность, — стекло не разобьётся. Пока есть люди, которые ищут прозрачность, — свет не погаснет. Пока есть люди, которые ценят заботу, — двери не закроются.

Дворец стоит не на фундаменте из кода. Он стоит на доверии.

А доверие — это единственный материал, который не бьётся.
