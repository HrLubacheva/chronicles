# 0.8.2 Вопросы, которые меняли всё

Они говорили не как заказчик с исполнителем. Они говорили как два строителя, которые смотрят на одно стекло и видят разное.

— Стоит ли разделить файлы? — спрашивал архитектор.
— Да, но не для скорости. Для понимания. Чтобы открыл папку — и сразу видно, где что.

— Почему страница тормозит?
— Пять наблюдателей следят за каждым изменением. Давай сделаем одного.

— А что насчёт дубликатов?
— Шесть копий одного защитника. Три мастера цен. Два провожатых. Давай оставим по одному.

Каждый вопрос был как луч света — высвечивал то, что можно улучшить. Каждый ответ — как новый камень в фундамент.
